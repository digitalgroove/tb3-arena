# TB3 Gazebo Package

tb3_gazebo is a Gazebo simulation with turtlebot 3 for recognizing AR Tags.

## Installation

### Requirements
* Joy Package


`$ sudo apt-get install ros-kinetic-joystick-drivers`

## Usage

```
$ roslaunch tb3_gazebo tb3_gazebo.launch
```
